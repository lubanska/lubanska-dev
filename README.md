## Installation

Instructions on how to set up the project locally. For example:

```bash
# Clone the repository
git clone https://github.com/lubanska/lubanska-dev.git

# Change to the project directory
cd lubanska-dev

# Install dependencies
npm install

# Start the project
npm run dev
```
